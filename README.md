# Bowen Patio Cover — 業務管理系統

A business management system for Bowen Aluminum / Bowen Patio Cover.

## Features
- 📦 **Product Library** — 104 SKUs with dimensions and weight
- 🧮 **Container Load Planner** — Smart 3D packing for 40' HQ containers
- 📋 **Order Management** — Create and track container orders
- 📊 **Load Map** — Top view and side view with real stacking visualization

## Live App
👉 **https://salehengmei-cpu.github.io/bowen-system/**

## How to use
Open the link above on any device — iPhone, Android, desktop. No install needed.

## Updating the app
To update, replace `index.html` with the new version and push to GitHub.
